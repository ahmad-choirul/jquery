Plugins jQuery del curso


SliderJS
Un slider o pase de diapositivas sencillo
http://www.slidesjs.com/ 

FlexSlider
Un slider o pase de diapositivas con un mayor número de opciones de configuración
http://flexslider.woothemes.com/ 
Plugin alternativo:
http://dev7studios.com/nivo-slider/ 

ImageLens
Incluye un efecto lupa a tus imágenes
http://www.dailycoding.com/Posts/imagelens__a_jquery_plugin_for_lens_effect_image_zooming.aspx 

Backstretch
Incorpora fácilmente imágenes que se adaptan automáticamente al tamaño de la ventana del navegador
http://srobbin.com/jquery-plugins/backstretch/ 

ScrollUp
Incluye de manera sencilla un botón para volver al top de tu página web.
http://markgoodyear.com/2013/01/scrollup-jquery-plugin/ 

AnimateScroll
Incluiremos animaciones de scroll para desplazarnos entre secciones de una página web
http://plugins.compzets.com/animatescroll/#
Plugin alternativo:
http://flesler.com/jquery/scrollTo/ 

OnePageScroll
Convierte tu página web en un gran pase de diapositivas
http://www.thepetedesign.com/demos/onepage_scroll_demo.html

Stellar.JS
Añade efectos parallax a tu página web
http://markdalgleish.com/projects/stellar.js/

JoyRide
Añade un asistente informativo a tu sitio web
http://zurb.com/playground/jquery-joyride-feature-tour-plugin 

Waypoints
Añade un asistente informativo a tu sitio web
http://zurb.com/playground/jquery-joyride-feature-tour-plugin 
Plugin alternativo:
http://ryanfunduk.com/jquery-tourbus/
