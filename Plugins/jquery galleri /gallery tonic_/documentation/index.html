XML Structure - top

The main content is loaded from a XML file, called portfolio.xml. You can also use a file with a different name- in the JavaScript section below is described how to do this.

In the XML file first you have to place the categories section- in this section you can specify all the portfolio categories. It looks like this:


	Graphic Design
	Web Design
	Photography

In order to create a category, you have to insert a category node. For this node you have to set an id attribute which should be unique for each category. The text in the node is the text that will be displayed.

After this comes the portfolio items section, where you can insert the portfolio items content. Each portfolio item should be described with the following structure:


	images/img1.jpg
	images/img2.jpg
	graphic,web
	This is description 1

Here is an explanation of each of the tags:

  • thumbnail- this is the path to the smaller thumbnail image, relatively to the html file
  • preview- this is the path to the bigger preview image, relatively to the html file
  • category- this is the category to which the item belongs. You have to insert the id of the category that you have set in the categories section above. You can insert multiple categories, by inserting their ids separated by commas (as shown on the example above)
  • description- this is the description of the item, it will be shown below the bigger preview image, when you click on the smaller image


JavaScript code - top

In order to activate the gallery with default settings for the div with id portfolio you have to insert this code in the head secttion of your html file:

    $('#portfolio').portfolioSetter();

Here are the default settings with explanations as comments to the right:

   xmlSource : 'portfolio.xml',  
			itemsPerPage: 12, 
			pageWidth: 960,  
			pageHeight:430, 
			itemMargin:30,  
			showCategories: true,  
			allText: 'ALL', 
			easing: 'easeOutExpo', 
			animationSpeed: 800, 
			navButtonWidth:21, 
			wavyAnimation:false,
			pageWrapperClass: 'page_wrapper',  
			navigationId: 'portfolio_pagination',  
			categoriesId: 'portfolio_categories', 
			itemClass: 'portfolio_item'

Here are the explanations of each of them:

  • itemsPerPage- the number of items per page. If you don't need pagination, just set a number that is bigger than the count of all your items
  • xmlSource-the path to the XML file from which the data is exctracted, relatively to the html file in which the gallery is initialized
  • pageWidth- the width of the page container
  • pageHeight-the height of the page container
  • itemMargin-margin of each of the portfolio items- the one that you set tp the portfolio_item class
  • showCategories- if set to false, the categories filter will be hidden
  • allText-the ALL text displayed before the categories names
  • easing-the animation easing, available values are:
    easeInQuad,easeOutQuad,easeInOutQuad,easeInCubic,easeOutCubic,easeInOutCubic,easeInQuart, easeOutQuart,easeInOutQuart,easeInQuint,easeOutQuint,easeInOutQuin,easeInSine,easeOutSine, easeInOutSine,easeInExpo,easeOutExpo,easeInOutExpo,easeInCirc,easeOutCirc,easeInOutCirc, easeInElastic,easeOutElastic,easeInOutElastic,easeInBack,easeOutBack,easeInOutBack,easeInBounce, easeOutBounce,easeInOutBounce
  • animationSpeed-the speed of the animation of the pagination
  • navButtonWidth-the width of the pagination button- if you change it in the CSS file, you have to change it here too, so that the pagination buttons will be centered
  • wavyAnimation-if set the true, all the elements will fade in consecutively with a wavy effect
  • pageWrapperClass-the class of the div that wraps the items in order to set a page
  • navigationId-the ID of the pagination div
  • categoriesId-the ID of the categories div
  • itemClass-the class of the div that wraps each portfolio item data


Examples

Getting the data from a different XML file:

 $('#portfolio').portfolioSetter({xmlSource:'portfolio2.xml'});

Enable the wavy animation:

 $('#portfolio').portfolioSetter({wavyAnimation:true});

Change the number of items per page to 10:

 $('#portfolio').portfolioSetter({itemsPerPage:10});

Read from different file, hide the category filter and change the animation easing :

 $('#portfolio').portfolioSetter({
 									   xmlSource:'portfolio2.xml', 
 									   showCategories:false, 
 									   easing:'easeInExpo'
 									 });

Loading the gallery with its default settings (you can use this code and just change the values you want):

 $('#portfolio_wavy').portfolioSetter({
							 			xmlSource : 'portfolio.xml',  
										itemsPerPage: 12, 
										pageWidth: 960,  
										pageHeight:430, 
										itemMargin:30,  
										showCategories: true,  
										allText: 'ALL', 
										easing: 'easeOutExpo', 
										animationSpeed: 800, 
										navButtonWidth:21, 
										wavyAnimation:false,
										pageWrapperClass: 'page_wrapper',  
										navigationId: 'portfolio_pagination',  
										categoriesId: 'portfolio_categories', 
										itemClass: 'portfolio_item'
						 			});
